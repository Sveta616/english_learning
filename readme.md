# English Learning Platform

Репозиторий содержит веб-приложение для изучения английского языка. Ниже приведена техническая спецификация и инструкция по развертыванию проекта.

## Технологический стек

### Backend & Database
* **PHP:** 8.2+ (используются расширения `mysqli`, `json`, `mbstring`)
* **MySQL:** 8.0+ (версия сервера `8.0.35 - MySQL Community Server`)
* **Клиент БД:** `libmysql - mysqlnd 8.2.18`
* **Управление:** phpMyAdmin 5.2.1 (стабильная версия 5.2.3)
* **Кодировка:** `UTF-8 Unicode (utf8mb4)`

### Frontend
* **Языки:** HTML5, CSS3
* **Скрипты:** JavaScript (ES6+)

### Окружение
* **Сервер:** Apache 2.4+
* **Модули:** обязательная поддержка `mod_rewrite`

---

## Требования к системе
Для корректной работы приложения убедитесь, что в вашей системе установлены:
1.  **PHP** 8.2 или выше.
2.  **MySQL** 8.0 или выше.
3.  **Apache** 2.4 с активированным `mod_rewrite`.
4.  **Git** (для управления версиями).

---

## Установка проекта

### 1. Клонирование репозитория
Откройте терминал и выполните следующие команды:

```
git clone https://github.com/Sveta616/english_learning.git
```

### 2. Настройка веб-сервера
Заранее требуется что-бы на компьютере был установлен локальный сервер, в нашем случае рекомендуем использовать OSPanel, в новых версия нужно отдельно скачивать phpMyAdmin, PHP и настраивать сам локальный сервер.

Для Apache:
 - Настроить виртуальный хост, указывающий на корневую директорию проекта
 - Убедиться, что включен модуль mod_rewrite
 - В файле .htaccess уже настроены необходимые правила

Для OpenServer (Windows):
 - Поместите папку проекта в OpenServer/home/
 - Переименуйте папку в english-learning (если скачалась под другим именем)
 - Запустите OpenServer и выберите домен english-learning

### 3. Настройка базы данных. 
Использование готового дампа english_learning.sql
Создайте базу в phpMyAdmin базу данных с названием english_learning:

```
CREATE DATABASE english_learning CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```
Импортируйте данные из прикрепленного sql скрипта

### 4. Конфигурация проекта
Настройка подключения к БД
Отредактируйте файл config.php под свои данные, если нужно замените подключения и константы конфига, если у вас реализуется своё подключение к базе данных и перенесенным из дамп-файла данным.
